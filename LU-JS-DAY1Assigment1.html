LU-JAVASCRIPT DAY1 - ASSIGNMENT 1 -The various methods in console function

console.log() - It is the display function which displays output in the console screen, eg:console.log("message");

console.error() - It is the display function which displays an error message to the console eg:console.error("message");

console.warn() - It is the function which Outputs a warning message to the console eg:console.warn("message");

console.table() - It is the display function which displays tabular data as a table eg:console.table("message");

console.time() - It is the display function which displays time taken by an operation to execute eg:console.time();

console.End() - It is the display function which displays finish time take by an operation to execute eg:console.timeEnd();
