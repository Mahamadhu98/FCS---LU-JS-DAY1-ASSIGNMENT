LU-JAVASCRIPT DAY1 - ASSIGNMENT 2- To display the var , let , const

var : It is a container which stores a value and it is mutable , it is accesible throughtout the program eg: var variable_name

let : It is a container which stores a value and it is mutable and most importanly the lifetime of 'let' is within a block of code. eg let let_variable_name

const : It is a container which stores a value and it is not mutable , it is accesible throughtout the program . eg : constant constant_value = 3.14
